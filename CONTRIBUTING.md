# Contributing to MCP Google Workers

## Welcome to the project!

We're glad you're interested in contributing to the MCP Google Workers project. This document provides guidelines for contributing to this repository.

## How to contribute

1. Fork the repository
2. Create a new branch for your feature or bug fix
3. Make your changes and commit them
4. Submit a pull request with a clear description of your changes

## Code of Conduct

This project is open source and welcomes contributions from the community. We follow a simple code of conduct:

- Be respectful and inclusive
- Provide constructive feedback
- Follow the existing code style
- Write clear commit messages
- Test your changes before submitting a pull request

## Development setup

1. Clone the repository:
   ```
   git clone https://github.com/your-username/mcp-google-workers.git
   cd mcp-google-workers
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Build the project:
   ```
   npm run build
   ```

## Reporting Issues

If you find a bug or have a feature request, please create an issue on GitHub with a clear description of the problem or enhancement.
